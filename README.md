
# Install

## Requirements
- Python 3.7 (or later)
- pip

## On Linux, MacOS
```bash
pip install motoman
```

## On Windows
```cmd
py -m pip install motoman
```

or, 

```bash
pip install --index-url https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple motoman
```
